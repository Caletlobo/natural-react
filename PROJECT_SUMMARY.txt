========================================
NATURAL REACT - PROJECT SUMMARY
========================================

COMPONENTS CREATED:
- App Component (Main component) - Monolithic component containing all sections


FEATURES IMPLEMENTED:

1. Hero Slider - Auto-rotating banner between 2 images every 3 seconds
2. Dark Mode Toggle - Theme switching button (üåô)
3. Product Cards - Display 6 seasonal ice cream products with BUY and READ MORE buttons
4. Buy Counter - Tracks number of items purchased per product
5. Video Player - Auto-playing muted video with mute/unmute controls
6. Instagram Post Gallery - Grid displaying posts with "Load More" functionality
7. Modal Alert - Caution/disclaimer modal shown on page load
8. Smooth Scroll - "Order Now" button scrolls to footer
9. Responsive Navigation - Menu with section links


STATE MANAGEMENT (useState):

- isDarkMode: Toggle dark/light theme (Boolean)
- showModal: Control modal visibility (Boolean)
- currentSlide: Track active hero slide 0-1 (Number)
- isVideoMuted: Track video audio state (Boolean)
- instaPosts: Store Instagram post images (Array)
- buyCounters: Store purchase count per product (Object)


REFS (useRef):

- videoRef: Reference to video element for mute control
- footerRef: Reference to footer for smooth scrolling


SIDE EFFECTS (useEffect):

1. Auto-slider - Rotates hero banner every 3 seconds
2. Modal Timer - Shows caution modal after 1 second on page load


EVENT HANDLERS (Functions):

- toggleTheme(): Switch dark mode
- handleBuy(productId): Increment product buy counter
- handleReadMore(description): Show product description alert
- loadMorePosts(): Add 5 more Instagram posts
- scrollToFooter(): Smooth scroll to footer
- toggleVideoMute(): Toggle video audio
